A simple text & markdown editor for Windows.

TextStyl is a simple plaintext & markdown editor and formatter made for Windows 10 using the UWP (Universal Windows Platform) framework.

Features

  • Plaintext creation and formatting.
  • Markdown creation and styling.
  • Easy-to-use graphical editing system.
  • Create notes that can be pinned to your desktop for quick reference.
  • Ability to encrypt notes using PGP encryption.

Platforms

Right now, TextStyl is only available on Windows 10, but I have future plans to port it to other platforms (macOS, Linux, etc.) if it seems to be in demand.

What is the signifigance of the name "TextStyl?"

"TextStyl" is a shorthand name for "Text Styler." Also, "TextStyl," when pronounced out loud, sounds very similar to "textile," hence the logo.

Where can I download TextStyl?

Right now, TextStyl is not available for download as it is still in a very early state of development. If you're curious and have some technical ability, you can visit the project's GitHub repo and clone it to your PC to build it. Make sure that you have Visual Studio with the UWP features installed before building.
