Welcome to ROAMER!

ROAMER is a simple adventure game made in QuickBasic for MS-DOS 6. You play as the Roamer: a mysterious character with unknown asperations lurking around a post-apocalyptic wasteland in search for an answer as to why the world as it was known was destroyed.

How To Run

NOTE: This hasn't been tested on a gen·u·ine MS-DOS installation, so your results may vary.

First, install DOSBOX and find an image of MS-DOS 6+.

Next, install MS-DOS on your DOSBOX installation. (I used this tutorial to do so.)

Once you've installed MS-DOS, copy the "ROAMER.EXE" file to your MS-DOS directory. (NOTE: You'll need to use an ISO viewer/editor program to do this.)

Lastly, use "cd" to navigate to where you've saved the "ROAMER.EXE" file to and simply type "ROAMER" and press enter. If everything works out, you should be able to play ROAMER!

Source Code

The full source and build code is available under the "SRC" directory. Make sure you have QuickBasic 7.1 installed before editing the source code. You can get a copy of QuickBasic 7.1 here.

To build the game, simply go to "Run > Make EXE File..." and use the following setting values to build ROAMER:

  • File Name: ROAMER.EXE
  • EXE Type: Stand-Alone EXE (/O)
  • Speed: 80x87 or Emulator Math (/FPi)
  • Target Environment: DOS or OS/2 Real Mode (/Lr)
  • Size/Capacity: Near Strings (/Fs)
  • Debug: [KEEP THIS BLANK]
  • Additional Options: [KEEP THIS BLANK]

...and then click "Make EXE". After a short while, you should have a built EXE of ROAMER in the same directory of the source code!
